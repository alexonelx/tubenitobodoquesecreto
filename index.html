Intercambio navideño.

Este proyecto nace para ordenar los intercambios navideños y hacer que la experiencia sea más clara para todos, desde el inicio hasta el final.

Cómo funciona

1) Te llega un link de tu familia (o grupo).
2) Entras, creas tu usuario y te unes al intercambio.
3) Agregas de 3 a 5 regalos con link + instrucciones (talla, color, etc.).
4) Cuando el admin inicia el sorteo, a cada persona le llega un mensaje por WhatsApp con a quién le toca regalar y la lista de regalos de esa persona.
